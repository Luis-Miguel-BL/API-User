# Para começar faça o seguinte:

<ol>
  <li>Crie um arquivo .env usando o .env.example como modelo </li>
  <li>Inicie o Docker com 'docker-compose build' e 'docker-compose up'</li>
  <li>Crie a database no banco de dados com o nome configurado no arquivo .env (recomendo o uso de um software como o DBeaver para realizar a conexão e criação da database)</li>
  <li>Instale as dependencias do projeto com 'yarn'</li>
  <li>Rode as migrations com 'yarn typeorm migration:run'</li>
  <li>Inicie o projeto com 'yarn dev'</li>
  <li>Utilize um software para testar os endpoints</li>
</ol>
